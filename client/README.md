# Budget App Front-end

### Installation

Clone repository:

```sh
git clone git@github.com:ely-alamillo/budget-app.git
```

Before running:

```sh
cd budget-app/client
npm install
```

## Run

### Local

```sh
npm start
```

- then go to `localhost:8000` to view the project

### Production build

```sh
npm run build
```
